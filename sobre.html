Sobre o CopaSync

O CopaSync é uma plataforma web criada para simplificar o acompanhamento da Copa do Mundo 2026, priorizando organização, clareza e personalização.

O que resolve

Reduz a poluição visual comum em plataformas esportivas e mostra o essencial: jogos, horários, grupos, chaveamento, favoritos e calendário.

Para quem foi criado

Para torcedores que desejam acompanhar a Copa de forma prática, seja no celular, tablet ou desktop, sem depender de uma interface pesada.

Dados locais

Nesta primeira versão, os dados ficam em arquivos JSON locais para garantir funcionamento no GitHub Pages sem backend ou API externa.

Melhorias futuras

API esportiva, placares em tempo real, login real, banco de dados, integração com Google Calendar, notificações e PWA instalável.
